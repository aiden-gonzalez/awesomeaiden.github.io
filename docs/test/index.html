section 00, design system

Design system

Every color, type voice, and motion token from design-tokens.css, rendered so the system can be signed off before any feature work begins.

01 Primary palette

--color-bone #f5f1ea Page background, light surfaces
--color-bone-warm #faf6ed Card surfaces (slightly lighter)
--color-bone-shade #efe9da Inset surfaces, headers, footers
--color-bone-line #c4b896 Borders, dividers, dashed lines
--color-forest #3a5a4a Structural, contour lines, labels, metadata
--color-forest-soft rgba(58, 90, 74, 0.55) Softened forest for secondary marks
--color-forest-faint rgba(58, 90, 74, 0.15) Faint forest for backgrounds, washes
--color-clay #8b3a2b Accent, waypoints, active states
--color-clay-tint #f5e4dc Clay background for "active" badges
--color-ink #2c2416 Body text, headings, hub center
--color-ink-soft #6b5d44 Secondary text, captions

02 Semantic aliases

Aliases reference primary tokens, change a primary, every alias follows.

  • --color-bg → --color-bone
  • --color-surface → --color-bone-warm
  • --color-surface-inset → --color-bone-shade
  • --color-border → --color-bone-line
  • --color-text-primary → --color-ink
  • --color-text-secondary → --color-forest
  • --color-text-tertiary → --color-ink-soft
  • --color-accent → --color-clay
